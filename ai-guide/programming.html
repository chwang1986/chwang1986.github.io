搭建 Python 环境，掌握 NumPy、Pandas 和 PyTorch，写出你的第一个神经网络

Python 环境搭建

工欲善其事，必先利其器

第一步：安装 Python

推荐从 python.org 下载最新稳定版（3.10 或以上）。安装时记得勾选 "Add Python to PATH"。安装完成后，在终端输入以下命令验证：

python --version
# 应该显示类似 Python 3.11.4 的版本号

第二步：安装 VS Code

VS Code 是目前最流行的代码编辑器，免费且功能强大。下载地址：code.visualstudio.com。安装后建议添加 Python 扩展和 Jupyter 扩展，方便编写和运行 Python 代码。

第三步：配置虚拟环境

虚拟环境可以隔离不同项目的依赖，避免版本冲突。在你的项目文件夹中运行：

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境（Windows）
venv\Scripts\activate

# 激活虚拟环境（macOS/Linux）
source venv/bin/activate

# 安装常用包
pip install numpy pandas matplotlib torch torchvision

第四步：验证 PyTorch 安装

python -c "import torch; print(torch.__version__); print(torch.cuda.is_available())"
# 如果显示版本号，说明安装成功
# cuda.is_available() 为 True 表示可以使用 GPU 加速

NumPy 和 Pandas 基础

AI 编程的两大基石：数值计算与数据处理

NumPy：高效的数值计算

NumPy 是 Python 科学计算的核心库，提供了高性能的多维数组对象 ndarray。AI 中几乎所有的数据最终都会变成 NumPy 数组或 PyTorch 张量。

import numpy as np

# 创建数组
a = np.array([1, 2, 3, 4, 5])
b = np.zeros((3, 3))          # 3x3 零矩阵
c = np.ones((2, 4))           # 2x4 全1矩阵
d = np.random.randn(3, 3)     # 3x3 随机数矩阵

# 基本运算
e = a + 10          # 每个元素加 10
f = np.dot(a, a)    # 点积
print(a.shape)      # 查看形状
print(a.mean())     # 求平均值

Pandas：数据处理与分析

Pandas 提供了 DataFrame 数据结构，让你像操作 Excel 一样操作数据。在 AI 项目中，数据预处理（清洗、转换、归一化）通常占 80% 的时间。

import pandas as pd

# 读取数据
df = pd.read_csv('data.csv')

# 查看数据
print(df.head())           # 前5行
print(df.info())           # 数据类型和缺失值
print(df.describe())       # 统计摘要

# 数据清洗
df = df.dropna()                      # 删除缺失值
df['age'] = df['age'].fillna(0)       # 填充缺失值
df = df[df['score'] > 60]             # 筛选数据

# 分组统计
print(df.groupby('category')['score'].mean())

PyTorch 入门

当前最流行的深度学习框架，简洁而强大

张量（Tensor）：PyTorch 的核心

张量是 PyTorch 的基本数据单位，可以理解成"加强版的 NumPy 数组"。它不仅能表示标量、向量、矩阵，还能表示更高维的数据。最重要的是，张量可以在 GPU 上运行，大大加速计算。

import torch

# 创建张量
x = torch.tensor([1.0, 2.0, 3.0])
y = torch.zeros(3, 3)
z = torch.randn(2, 3)       # 标准正态分布随机数

# 张量运算
a = x + 2
b = torch.matmul(z, z.T)    # 矩阵乘法

# 转到 GPU（如果可用）
if torch.cuda.is_available():
    x = x.cuda()
    print("张量已在 GPU 上")

# 和 NumPy 互转
import numpy as np
np_array = x.cpu().numpy()
torch_tensor = torch.from_numpy(np_array)

自动求导（Autograd）

神经网络的训练离不开梯度下降，而梯度需要计算导数。PyTorch 的 autograd 模块可以自动计算梯度——你只需要定义前向传播，反向传播交给它。

import torch

# 创建需要梯度的张量
x = torch.tensor(2.0, requires_grad=True)

# 定义计算（前向传播）
y = x ** 2 + 3 * x + 1

# 反向传播，自动计算梯度
y.backward()

# dy/dx = 2x + 3，当 x=2 时，梯度为 7
print(x.grad)   # 输出: tensor(7.)

这个简单的例子展示了 PyTorch 的精髓：你定义计算图，PyTorch 自动帮你求导。在复杂的神经网络中，这一特性让开发者从繁琐的求导工作中解放出来。

第一个神经网络实战：手写数字识别

经典的 MNIST 数据集，AI 界的"Hello World"

我们将构建一个简单的神经网络，识别 0-9 的手写数字。数据集包含 60,000 张训练图片和 10,000 张测试图片，每张是 28x28 像素的灰度图。

完整代码

import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import datasets, transforms
from torch.utils.data import DataLoader

# 1. 数据准备
transform = transforms.Compose([
    transforms.ToTensor(),           # 转为张量，并归一化到 [0,1]
    transforms.Normalize((0.5,), (0.5,))  # 标准化到 [-1,1]
])

train_data = datasets.MNIST(root='./data', train=True,
                            download=True, transform=transform)
test_data  = datasets.MNIST(root='./data', train=False,
                            download=True, transform=transform)

train_loader = DataLoader(train_data, batch_size=64, shuffle=True)
test_loader  = DataLoader(test_data,  batch_size=64, shuffle=False)

# 2. 定义神经网络
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.flatten = nn.Flatten()
        self.fc1 = nn.Linear(28*28, 128)   # 输入层 → 隐藏层
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(128, 10)      # 隐藏层 → 输出层（10个数字）

    def forward(self, x):
        x = self.flatten(x)
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)
        return x

model = Net()

# 如果有 GPU，把模型移到 GPU
if torch.cuda.is_available():
    model = model.cuda()

# 3. 定义损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 4. 训练循环
epochs = 5
for epoch in range(epochs):
    total_loss = 0
    for images, labels in train_loader:
        if torch.cuda.is_available():
            images, labels = images.cuda(), labels.cuda()

        optimizer.zero_grad()      # 清空旧梯度
        outputs = model(images)    # 前向传播
        loss = criterion(outputs, labels)  # 计算损失
        loss.backward()            # 反向传播
        optimizer.step()           # 更新权重

        total_loss += loss.item()

    print(f"Epoch {epoch+1}/{epochs}, Loss: {total_loss/len(train_loader):.4f}")

# 5. 测试准确率
model.eval()
correct = 0
total = 0
with torch.no_grad():
    for images, labels in test_loader:
        if torch.cuda.is_available():
            images, labels = images.cuda(), labels.cuda()
        outputs = model(images)
        _, predicted = torch.max(outputs, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f"测试准确率: {100 * correct / total:.2f}%")
# 通常可以达到 97% 以上

代码解析

  • 数据加载：torchvision.datasets 自动下载 MNIST 数据，DataLoader 把数据分成小批量（batch），方便高效训练。
  • 网络结构：我们的网络有 3 层：输入层（784=28x28 个神经元）、隐藏层（128 个神经元，带 ReLU 激活）、输出层（10 个神经元，对应 0-9）。
  • 损失函数：交叉熵损失（CrossEntropyLoss）适合分类任务，它同时包含了 Softmax 激活和负对数似然损失。
  • 优化器：Adam 是目前最常用的优化器，它结合了动量和自适应学习率，收敛速度快且稳定。
  • 训练循环：每个 epoch 遍历所有训练数据，执行"前向传播 → 计算损失 → 反向传播 → 更新权重"四步。
