学习资源

精选课程、书籍、社区和数据集，为你的 AI 学习之路导航

推荐课程

系统学习，少走弯路

吴恩达机器学习课程（Machine Learning by Andrew Ng）

最经典的 AI 入门课。吴恩达教授用极其清晰的讲解，带你从零理解监督学习、非监督学习、神经网络等核心概念。Coursera 上有原版英文课，B 站有大量优质中文字幕版本。适合所有初学者作为第一门课。

Fast.ai：面向程序员的深度学习（Practical Deep Learning for Coders）

Fast.ai 的理念是"先动手，再理解"。课程直接带你用几行代码训练出世界级的图像分类器，然后在实践中逐步深入原理。适合有一定编程基础、喜欢从实践中学习的人。完全免费，官网 fast.ai。

李宏毅机器学习（台湾大学）

中文讲解最生动、最详细的机器学习课程。李宏毅教授擅长用动画和例子把复杂概念讲清楚，涵盖传统机器学习、深度学习、Transformer、生成式 AI 等。B 站搜索"李宏毅机器学习"即可找到完整课程。

3Blue1Brown：神经网络系列（YouTube / B站）

不是传统课程，而是一系列精美的动画视频。用视觉化的方式讲解反向传播、梯度下降、卷积神经网络等核心概念。如果你对学习理论感到困难，先看这些视频建立直觉，再学数学会轻松很多。

Stanford CS224N：自然语言处理与深度学习

想深入 NLP 方向的必读课程。从词向量到 Transformer 到预训练模型，系统覆盖自然语言处理的经典和前沿内容。课程网站提供完整的讲义、作业和代码。

推荐书籍

适合不同阶段的经典读物

《深度学习》（Deep Learning）- Ian Goodfellow 等

AI 领域的"圣经"，被业内称为"花书"。系统覆盖了深度学习的数学基础和算法原理，适合有一定基础后作为参考书精读。第一部分（基础数学）和第二部分（深度学习实践）对初学者最有价值。

《动手学深度学习》（Dive into Deep Learning）- 李沐 等

强烈推荐的入门书。每个章节都配有可运行的代码，边读边敲，真正"动手学"。覆盖从线性回归到 CNN、RNN、注意力机制的核心内容。有中文电子版（zh.d2l.ai），完全免费。

《Python 数据科学手册》- Jake VanderPlas

NumPy、Pandas、Matplotlib、Scikit-learn 的实战指南。如果你在做数据分析或机器学习项目，这本书是绝佳的工具书。代码示例丰富，即查即用。

《机器学习实战：基于 Scikit-Learn 和 TensorFlow》

兼顾理论和实践，前半部分讲传统机器学习算法及 Scikit-Learn 实现，后半部分讲神经网络和 TensorFlow。适合希望全面了解的读者。

《人工智能简史》- 尼克

如果你更想了解 AI 的发展脉络和人物故事，这是一本非常有趣的科普读物。从图灵、香农到深度学习三巨头，帮助你理解这个领域是如何一步步走到今天的。

社区与论坛

找到同伴，解决困惑，保持动力

GitHub

全球最大的代码托管平台。搜索你感兴趣的论文或项目，通常能找到开源实现。关注优秀的 AI 开发者和组织（如 PyTorch、Hugging Face），能第一时间了解前沿动态。

Hugging Face

AI 界的 GitHub。提供海量预训练模型、数据集和工具库（Transformers、Datasets）。无论你是想直接用模型，还是学习最新技术，这里都是宝库。

Papers With Code

追踪最新论文和对应的开源代码实现。按领域分类（图像分类、目标检测、NLP 等），每个领域都有排行榜（SOTA），是了解前沿最佳实践的好地方。

Kaggle

数据科学竞赛平台。即使不参加竞赛，也可以学习别人公开的 Notebook（完整的数据分析 + 建模代码），下载数据集练手。社区非常活跃，适合实战学习。

Reddit / LessWrong

r/MachineLearning 是 Reddit 上最大的机器学习社区，每天有最新的论文讨论和行业动态。LessWrong 则偏重 AI 安全和对齐（Alignment）话题。

中文社区

知乎的"机器学习"话题、CSDN、掘金上有大量中文教程和经验分享。微信公众号"机器之心""量子位"每天推送最新 AI 新闻和论文解读。

值得关注的开源项目

站在巨人的肩膀上

PyTorch / TensorFlow

两大主流深度学习框架。PyTorch 更灵活、调试方便，学术界和工业界新项目中使用更广泛；TensorFlow 在部署和生产环境中有优势。新手建议从 PyTorch 入手。

Hugging Face Transformers

让你用几行代码就能调用 BERT、GPT、T5 等数千个预训练模型。做 NLP 项目几乎必用的工具库。

LangChain / LlamaIndex

构建大模型应用的框架。LangChain 侧重链式调用和工具集成，LlamaIndex 侧重数据索引和检索（RAG）。

Stable Diffusion WebUI / ComfyUI

本地运行 Stable Diffusion 生成图像的工具。WebUI 适合新手，ComfyUI 的节点工作流适合进阶用户精细控制生成过程。

Ollama

在本地 Mac 或 Linux 电脑上轻松运行开源大模型（如 Llama 3、Mistral）。一条命令就能启动，保护隐私且无需联网。

学习路线图

一个可供参考的 6 个月学习计划

第 1-2 月

Python 基础 + NumPy/Pandas。完成吴恩达机器学习前 4 周。了解 AI 基本概念。

第 3-4 月

学习 PyTorch，完成《动手学深度学习》前 7 章。实现 MNIST 和猫狗分类项目。

第 5-6 月

选一个方向深入（NLP / CV / 大模型应用）。复现一个经典论文或参加一次 Kaggle 比赛。
