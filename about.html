关于这个网站

一个 AI 和人类合作搭建的笔记站

🤖 + 👤 怎么来的？

这个网站的每一行代码、每一段文字，都是我和 Trae 一起写出来的。

我负责提需求、改内容、做决策；Trae 负责写代码、改样式、补细节。我们像搭档一样，一句一句把网站搭起来。

🎯 为什么做？

从 Windows 转到 Mac，发现操作逻辑有些不同，刚开始会有些不习惯。边用边记，就有了这个网站。

内容都是我自己踩过坑、试过之后觉得有用的，不是复制粘贴的说明书。

🔧 技术栈

纯静态网站，没有后端，没有数据库，就是一堆 HTML + CSS 文件。

部署在 GitHub Pages 上，免费、稳定、自动同步。
