环境部署

把 Mac 变成 AI 开发机的步骤

第一步：装 Homebrew

Mac 上最好用的包管理器，装软件、装环境都用它。

打开终端，粘贴这行命令，回车：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

装完之后运行这两行，把 Homebrew 加到环境变量：

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
验证：运行 brew --version，看到版本号就说明装好了。

第二步：装 Python

Mac 自带 Python，但版本老。我们用 Homebrew 装最新的：

brew install python

装完之后验证：

python3 --version
pip3 --version
注意：以后用 python3 和 pip3，不是 python 和 pip。

第三步：装 Git

Mac 自带 Git，但版本可能不够新：

brew install git

配置你的名字和邮箱：

git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

第四步：装 Node.js（可选）

有些 AI 工具的前端界面需要 Node：

brew install node

第五步：创建虚拟环境（推荐）

不同项目需要的 Python 库版本不一样，用虚拟环境隔离：

# 创建虚拟环境
python3 -m venv ~/myenv

# 激活
source ~/myenv/bin/activate

# 退出
deactivate
建议：每个项目建一个虚拟环境，避免库版本冲突。

第六步：装常用 Python 库

pip3 install numpy pandas matplotlib jupyter requests

这些都是 AI 开发最常用的基础库。
