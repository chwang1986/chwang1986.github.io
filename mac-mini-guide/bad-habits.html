好习惯 vs 坏习惯

用对方法，Mac 越用越顺手

✅ 好习惯

1. 用 Spotlight 找一切

按 Cmd + Space，输入应用名、文件名，甚至算个数学题。

比鼠标点开 Finder 找快十倍。

2. 定期清理桌面

桌面文件太多会拖慢系统。每周整理一次，把文件归类到「文稿」里的对应文件夹。

3. 用标签分类文件

Finder 里右键文件 → 标签，给文件打色标。找文件时一眼就能定位。

4. 学会用预览

Finder 里选中文件按 Space，不用打开软件就能看内容。图片、PDF、视频都行。

5. 多桌面分工

四指上滑打开调度中心，新建桌面。工作、娱乐、学习分开，互不干扰。

❌ 坏习惯

1. 所有文件丢桌面

后果：桌面文件太多会让 Finder 变卡，开机变慢，找东西像大海捞针。

改：桌面只放临时文件，当天处理完就归档。

2. 不关应用只关窗口

后果：点左上角的红色叉只是关窗口，应用还在后台吃内存。Mac 风扇狂转、电池尿崩，多半是这个原因。

改：不用了按 Cmd + Q 彻底退出。

3. 从不清理废纸篓

后果：删了的文件其实还在硬盘里占空间，SSD 空间满了性能暴跌。

改：右键废纸篓 → 清倒废纸篓，至少一个月一次。

4. 安装来历不明的软件

后果：Mac 不是百分百安全，不明来源的软件可能带病毒或挖矿程序。

改：尽量用 App Store 或官网，别点百度下载站的「高速下载」。安裝打不开时去「系统设置 → 隐私与安全性」手动放行。

5. 忽视系统更新

后果：安全漏洞不补，新功能用不了，软件兼容性出问题。

改：系统设置 → 通用 → 软件更新，打开自动更新。大版本更新可以等一个月再升，小版本补丁及时装。
