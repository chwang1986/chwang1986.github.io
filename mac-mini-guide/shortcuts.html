快捷键

每天省 10 分钟的按键组合

最常用（必背）

快捷键作用
Cmd + C复制
Cmd + V粘贴
Cmd + Z撤销
Cmd + A全选
Cmd + F查找
Cmd + W关闭当前窗口
Cmd + Q退出应用
Cmd + Tab切换应用

文件操作

快捷键作用
Cmd + N新建窗口
Cmd + O打开文件
Cmd + S保存
Cmd + Shift + S另存为
Cmd + Delete移到废纸篓
Cmd + Shift + N新建文件夹
Space快速预览文件

截图（超实用）

快捷键作用
Cmd + Shift + 3全屏截图
Cmd + Shift + 4区域截图
Cmd + Shift + 4 + Space窗口截图
Cmd + Shift + 5截图工具栏（录屏也在这）
Cmd + Ctrl + Shift + 3截图到剪贴板（不存文件）

快捷键作用

Cmd + H隐藏当前应用
Cmd + Option + Esc强制退出
Cmd + `同应用切换窗口
