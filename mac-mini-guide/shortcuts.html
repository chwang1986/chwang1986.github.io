快捷键

每天省 10 分钟的按键组合

最常用

快捷键作用
Cmd + C复制
Cmd + V粘贴
Cmd + X剪切（仅限文字）
Cmd + Z撤销
Cmd + Shift + Z重做
Cmd + A全选
Cmd + F查找
Cmd + W关闭当前窗口
Cmd + Q退出应用
Cmd + Tab切换应用
Cmd + `同应用切换窗口
Cmd + H隐藏当前应用
Cmd + M最小化窗口

文件操作

快捷键作用
Cmd + N新建窗口
Cmd + O打开文件
Cmd + S保存
Cmd + Shift + S另存为
Cmd + Delete移到废纸篓
Cmd + Shift + N新建文件夹
Cmd + I显示文件简介
Cmd + D复制所选文件
Space快速预览文件

截图

快捷键作用
Cmd + Shift + 3全屏截图，保存到桌面
Cmd + Shift + 4区域截图，保存到桌面
Cmd + Shift + 4 + Space窗口截图，保存到桌面
Cmd + Shift + 5截图工具栏（支持录屏）
Cmd + Ctrl + Shift + 3全屏截图到剪贴板
Cmd + Ctrl + Shift + 4区域截图到剪贴板

系统

快捷键作用
Cmd + SpaceSpotlight 搜索
Cmd + ,当前应用偏好设置
Cmd + Option + Esc强制退出应用
Cmd + Shift + Q注销用户
Cmd + Option + Power睡眠
Ctrl + Power显示关机选项

文本编辑

快捷键作用
Cmd + ←跳到行首
Cmd + →跳到行尾
Option + ←跳到上一个单词
Option + →跳到下一个单词
Cmd + ↑跳到文档开头
Cmd + ↓跳到文档结尾
Option + Delete删除前一个单词
Cmd + Delete删除到行首

浏览器（Safari / Chrome）

快捷键作用
Cmd + T新建标签页
Cmd + W关闭标签页
Cmd + Shift + T恢复刚关闭的标签页
Cmd + L定位到地址栏
Cmd + R刷新页面
Cmd + [返回上一页
Cmd + ]前进下一页
Cmd + 数字切换到第 N 个标签页

终端

快捷键作用
Ctrl + C终止当前命令
Ctrl + A光标移到行首
Ctrl + E光标移到行尾
Ctrl + U删除整行
Ctrl + R搜索历史命令
Ctrl + L清屏
Tab自动补全命令/路径
