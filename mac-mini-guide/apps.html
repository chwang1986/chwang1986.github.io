软件推荐

按场景分类，供你参考

🧑‍💻 开发工具

VS Code

写代码必备。免费，插件多，支持各种编程语言。App Store 里没有，需要去官网下载。

获取 官网下载

终端（系统自带）

Mac 自带的终端已经够用。如果觉得功能不够，可以在「终端 → 设置」里调整字体、配色和窗口大小。

🎨 效率工具

Xmind

思维导图工具，整理思路、做笔记很方便。有免费版，基础功能够用。

免费版可用

Typora

Markdown 编辑器，所见即所得。写文档、记笔记很顺手，导出 PDF 也方便。

付费 有试用版

系统自带备忘录 + 提醒事项

苹果的备忘录和提醒事项其实挺好用的，iCloud 同步，手机电脑都能看。先试试自带的，不够用了再装别的。

🌐 浏览器

Safari

Mac 自带的浏览器，省电，和系统整合得好。日常使用完全没问题。

Chrome

兼容性最好，插件最全。有些网站 Safari 打不开时可以用它。

免费

💬 通讯办公

微信 / 钉钉 / 飞书

日常沟通必备，按你所在的公司或学校用的平台选择即可。

腾讯会议

远程会议，Mac 版摄像头和麦克风管理比较直观。

📚 学术研究

Zotero

文献管理工具，免费开源。可以收集论文、做笔记、生成引用格式。

免费

PDF Expert

PDF 阅读和标注工具，界面干净，标注功能顺手。

付费 有试用版

Obsidian

知识管理工具，用 Markdown 记笔记，支持双向链接。数据存在本地，隐私性好。

免费 个人使用免费
