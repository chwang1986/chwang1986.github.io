软件推荐

我装过觉得好用的，按场景分类

🧑‍💻 开发工具

VS Code

写代码必备。免费，插件多，AI 辅助编程的插件也很全。

获取 官网下载 / App Store

iTerm2

比系统自带的终端好用一百倍。分屏、搜索、配色都能自定义。

获取 官网免费下载

Cursor

AI 代码编辑器，内置 GPT-4，写代码时按 Tab 就能自动补全。

付费 有免费额度

🎨 效率工具

Raycast

Spotlight 的终极升级版。启动应用、搜索文件、剪贴板历史、窗口管理，一个软件全搞定。

免费 基础版完全够用

Snipaste

截图 + 贴图。截图后可以直接贴在屏幕上，写代码对照文档超方便。

免费

Itsycal

菜单栏小日历。点一下看日历和日程，比系统自带的方便。

免费

🌐 浏览器

Arc

新形态浏览器，标签页在左边，支持分屏、工作区、自动归档。用惯了回不去。

免费

Chrome

兼容性最好，插件最全。Arc 搞不定的网站用它。

免费

💬 通讯办公

微信 / 钉钉 / 飞书

日常沟通必备，Mac 版体验比 Windows 版好很多。

腾讯会议 / Zoom

远程会议，Mac 版摄像头和麦克风管理更直观。

🎵 媒体娱乐

IINA

视频播放器，免费开源，什么格式都能播，界面还好看。

免费

网易云音乐 / Spotify

听歌必备，Mac 版支持键盘媒体键控制。
