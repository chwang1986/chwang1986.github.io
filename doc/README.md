# 项目文档

## 项目说明

本项目是人工智能（AI）与人类合作搭建的静态网站，部署在 GitHub Pages 上。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 网站首页 |
| `start.sh` | 本地预览服务器（会自动打开浏览器） |
| `push-to-github.sh` | GitHub 快速推送脚本 |

## 使用方式

## 设计风格

### 1. 视觉风格
- **简单、科技、浅色**：以白色和浅灰色为底色，搭配简洁的蓝色或青色作为强调色，避免过多装饰，突出内容的科技感。

### 2. 内容风格
- **类似笔记**：排版松散、留白充足，像手写笔记一样自然。
- **配色鲜明**：标题、重点词汇使用高饱和度的强调色（如亮蓝、青绿），与浅色背景形成对比。
- **语言简短、口语化**：避免长句和书面语，用短句、口语化的表达，降低阅读门槛。

### 3. 关于页面
- 网站必须包含一个明确的 `about.html`（或关于 section）。
- 在关于页面中清楚说明：**本网站是由人工智能（AI）与人类合作实现的**，体现人机协作的理念。
